How to use Ostos

Introduction

"Ostos" on ostoslista-applikaatio. Se on artesaanituote, joka on kehitetty pitkän ajan kuluessa, analysoiden päivittäisten ostosten tekoa huolellisesti.

Sovellus muistaa artikkelit, jotka olet sille syöttänyt, eikä kerro kenellekkään muulle! Ei mainoksia eikä kauppaketjuyhteyksiä... eikä muitakaan yhteyksiä tällä hetkellä. Toivottavasti tästä on iloa sinulle!

Ostoslistasivu

Ensimmäinen sivu joka aukeaa on ostoslistasivu.

Lisää uusia artikkeleita pyyhkäisemällä vasemmalle, tai alasvetovalikosta Add new items by swithcing to search page on the right (on first item in the pulldown menu.
The items have states, indicated by the icon on the left of each shopping list item.
The state can be toggled like by clicking the state icon on the left.

  • The item is listed as to buy
  • The item has been bought
  • The item is listed as flagged

An item can also be hidden but then it is not shown in the list

The context menu has options to:

  • hide
  • flag
  • edit
  • increase quantity
  • decrease quantity
  • delete the item.

Add new items by swithcing to search page on the right on first item in the pulldown menu

Pulldown menu

There are options in Pulldown menu:

  • Enter to buy - add items to list. Same as swipe left / next page
  • Edit Shops - go to to the dialog page for adding shops
  • Settings - go to settings page. Few settings.
  • Help - go to this page

Navigation

Swipe navigation is supported; swipe left to add items, dialogs go first to search page and then detail edit and confirmation of the item added to the list. Swipe right to go back.

Search (Add) page

Type to the text field nex to , or pick your match from the shown list of buttons. Accept with Accept or swipe left.

Edit page

The page shows item details retrieved from database, if any. The user should edit the values to his liking, and the press Accept or swipe left.
