Einleitung

"Warenkorb" ist eine Einkaufsliste App. Es ist ein handgefertigtes Produkt, dazu man natürlichen Flüss des Einkaufs analysiert gehabt.

Die App merkt sich, was Sie eingegeben haben, aber nicht jemand anderes sagen! Keine Anzeigen oder Shop-Kette Verbindungen ... keine Anschlüsse an alle. Ich wünsche Ihnen eine angenehme Benutzererfahrung!

Einkaufsliste - erste Seite

Die erste Seite enthält die Einkaufsliste.

Hinzufügen neue Artikel der Suchseite auf der rechten Seite wischen, oder wählen Sie die erste Option im Schiebemenü. Drehen Seite.
Die Artikeln haben Staaten, die durch das Symbol auf der linken Seite jeder Artikel angezeigt.
Der Staat kann wie den Zustand Symboldurch Anklicken auf der linken Seite umgeschaltet werden.

  • zu kaufen
  • gekauft
  • markiert

Ein Artikel kann auch der Staat 'versteckt' haben, aber dann ist es nicht in der Liste angezeigt.

Das Kontextmenü bietet Optionen:

  • verstecken
  • markieren
  • bearbeiten
  • ansteigen Menge
  • abnehmen Menge
  • Artikel löschen

Hinzufügen neuer Artikel auf Suchseite auf der rechten Seite oder durch den ersten Punkt im Schiebemenü

Schiebemenü

Optionen in Schiebemenü

  • Suche und ergänze - gleich wie links wischen
  • Versteck gekaufte - so werden alle gekaufte Artikeln versteckte
  • Handeln bearbeiten - die in der Lite
  • Einstellungen
  • Anweisung - diese Seite

Navigation

Navigation zu wischen functioniert. Links wischen artikel suchen oder ergänzen, dann Artikel bearbeiten und akzeptieren. Rechts wischen macht's zurück.

Suschseite

Eingeben in das Textfeld neben dem , oder tippen Sie aus der Liste auszuwählen. Akzeptieren Sie mit "Akzeptieren" oder links wischen.

Bearbeitsseite

Die Seite zeigt Nähere von dem Artikel, abgeruft aus dem Datenbank, falls vorhanden. Der Benutzer sollte die Werte rechte machen, und fertige Akzeptieren oder links wischen.

Seite für bearbeitung die Läden

Die Seite kann durch Schiebemenü eingegeben werden, ggf. Eine Liste von Ladennamen wird angezeigt. Jede Zeile enthält Kontextmenü , in dem Bearbeiten oder Löschen ausgewählt werden können. Die Namen werden auf der Stelle bearbeitet, daran keine weitere Seite geöffnet wird. Fertige Akzeptieren oder links wischen.

Ein neues Geschäft Name kann mit Schiebemenü "Neuer Laden Hinzüfugen" hinzugefügt werden, die eine Dialogseite wird geöffnet. Fertige annehmen oder Links wischen.

Die Liste von die Läden und seine Bedeutung

Die Ladennamen werden als Filter auf der Einkaufsliste verwendet. Mit dem Platzhalter - *, Stern oder Sternchen, werden alle Artikeln auf der Einkaufsliste zeigen. Each shopping list item has Shop property which can be selected from the Shop list. Jeder Artikel hat "Laden" Eigenschaft, die aus der Ladenliste kann ausgewählt werden.
